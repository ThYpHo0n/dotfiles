# Dotfiles
```
 hyper.is   > hyper.is terminal settings
 zsh        > oh-my-zsh plugins and themes
 ```

# Usage
I use [stow](https://www.gnu.org/software/stow/) to manage my dotfiles
```
git clone https://github.com/aeolyus/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow zsh
```
